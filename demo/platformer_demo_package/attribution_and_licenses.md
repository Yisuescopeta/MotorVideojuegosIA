# Attribution and licenses

## Visual assets

Los assets visuales seleccionados pertenecen al estilo y naming del `Kenney Platformer Pack`, aquí referenciados a través del mirror público `Barney241/Jump2D`.

Uso previsto en este paquete:

- jugador: `alienBlue_*`
- tiles de suelo/plataforma: `grass*`
- coleccionable: `coinGold.png`
- hazard: `spikes.png`

### Situación de licencia

La intención de selección ha sido priorizar assets tipo Kenney con licencia muy permisiva. Antes de usar el demo en una entrega externa conviene verificar de nuevo el `License.txt` original del pack exacto descargado por el script, pero la selección está hecha pensando en un set CC0 / uso permisivo de prototipo.

## Audio

Los cuatro sonidos del prototipo se generan localmente desde `fetch_selected_assets.py`. Por tanto:

- no requieren atribución a terceros
- son seguros para prototipo
- sirven como placeholder técnico para validar el runtime de audio

## Recomendación práctica

Si quieres máxima trazabilidad legal dentro del repo, tras ejecutar el script añade también el `License.txt` original del pack visual descargado en esta misma carpeta o dentro de `assets/optional/`.
